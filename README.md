## Current Project — RedForce AI

**Founder and Principal Architect**

RedForce AI is an enterprise-grade, autonomous security testing platform currently under active development. It is designed with a hard constraint: zero data egress. All inference runs locally.

**Architecture highlights:**
- 13-engine testing framework covering OWASP Top 10:2025, Mobile, API, and LLM security
- Local LLM inference via Ollama — no data leaves the environment
- Self-improving learning loop stored in a versioned PostgreSQL standards registry
- Go API backend, React/Vite frontend, Python scanning engines, Docker orchestration
- Proprietary vulnerability knowledge base — 249 CWEs, 92 OWASP test cases

RedForce AI is being built to demonstrate that enterprise AI security tooling does not require cloud dependency, vendor lock-in, or compromise on data sovereignty.
